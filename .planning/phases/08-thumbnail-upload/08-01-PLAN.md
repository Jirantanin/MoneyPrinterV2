**1. generate_metadata(self) -> dict:**

This method generates YouTube metadata using the LLM. It must:

- Guard: raise ValueError if self.episode_dir is not set
- Guard: raise FileNotFoundError if script.json does not exist in episode_dir
- Read script.json to get the 14 scenes (needed for context)
- Extract narrations from scenes 0 and 1 (intro + first act1 scene) as context_excerpt
- Call generate_text() with this prompt:

```
You are a YouTube metadata writer for long-form podcast videos.

Topic: {self.topic}
Opening narration: {context_excerpt}

Generate YouTube metadata for this podcast episode. Return ONLY valid JSON with these keys:
- "title": Engaging podcast title, max 90 characters, NO emoji, NO hashtags
- "description": 2-3 paragraph description (max 4000 chars) summarizing what the episode covers. End with 3-5 relevant hashtags on a new line.
- "tags": Array of 8-15 relevant tags as strings

Return ONLY the JSON object, no markdown fencing.
```

- Parse the LLM response as JSON. If parsing fails, try stripping markdown code fences (```json ... ```) and retry parse.
- If JSON parsing still fails, use fallback metadata:
  ```python
  metadata = {

      "description": f"A deep dive podcast episode about {self.topic}.",
      "tags": [self.topic.lower(), "podcast", "deep dive", "storytelling"],
  }
  ```
- Truncate title to 100 chars, description to 5000 chars
- Store result in self.metadata (dict with keys: title, description, tags)
- Write metadata to episode_dir/metadata.json
- Print the generated title
- Return the metadata dict

**2. generate_thumbnail(self) -> str | None:**

This method generates a thumbnail image using Gemini. It must:

- Guard: raise ValueError if self.episode_dir is not set

- If thumbnail.png already exists, print skip message and return output_path (resumability)
- Build the prompt: f"{self.style_prompt} A dramatic podcast cover image for an episode about: {self.topic}. Bold, cinematic composition. No text overlay."
- Call generate_image(prompt, output_path) from image_provider
- If result is None, print warning "Thumbnail generation failed" and return None
- Otherwise print success message and return output_path

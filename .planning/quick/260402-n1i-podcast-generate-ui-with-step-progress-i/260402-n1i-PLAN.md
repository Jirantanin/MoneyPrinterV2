```python
class Podcast:
    def __init__(self, topic: str = "") -> None:
        self.topic = topic
        self.episode_dir: str = ""   # Set by generate_script()
        self.metadata: dict = {}     # Set by generate_metadata()

    def generate_script(self, topic: str) -> list:
        """Returns list of 14 scene dicts with 'narration' and 'image_prompt' keys.
        Sets self.episode_dir to .mp/podcast_{slug}_{YYYYMMDD}/
        Writes script.json to episode_dir."""

    def generate_assets(self) -> None:
        """Generates scene_NN.png and scene_NN.wav for each of 14 scenes.
        Prints 'Generating assets for scene {i+1}/{total}...' to stdout.
        Prints 'Skipping scene {i+1}/{total} (already generated).' for resumable scenes."""

    def render(self) -> None:
        """Renders per-scene .mp4 clips then concatenates to final.mp4.
        Prints 'Rendering scene {i+1}/14...' and 'Concatenating clips into final.mp4...'"""

    def generate_metadata(self) -> dict:
        """Returns dict with keys: title, description, tags.
        Persists to metadata.json in episode_dir."""

    def generate_thumbnail(self) -> str | None:
        """Returns path to thumbnail.png or None on failure."""

    def upload(self) -> str | None:
        """Uploads final.mp4 + thumbnail to YouTube. Returns video URL or None."""
```

Create `src/podcast_server.py` — a FastAPI application that serves the podcast UI and orchestrates the pipeline via background threads with SSE progress streaming.

**Module-level state:**
- `episodes: dict[str, dict]` — keyed by episode_id, stores: `podcast` (Podcast instance), `status` (str: idle/running/done/error), `current_step` (int 0-5), `step_states` (list of 5 dicts with keys: name, status, message), `scenes` (list from generate_script), `metadata` (dict), `error` (str or None), `episode_dir` (str)
- `episode_events: dict[str, list]` — keyed by episode_id, list of SSE event dicts to be consumed by the stream endpoint

**Routes:**

1. `GET /` — Reads and returns `src/podcast_ui.html` using `FileResponse` or reads file content and returns `HTMLResponse`. Use `os.path.join(os.path.dirname(__file__), "podcast_ui.html")` to locate the file.

2. `POST /api/generate` — Accepts JSON body `{"topic": str}`. Creates a Podcast instance, generates a unique episode_id (use `f"podcast_{slug}_{datestr}"` matching Podcast.py pattern), stores in episodes dict, starts a background thread running `_run_pipeline(episode_id)`, returns `{"episode_id": episode_id}`.

3. `GET /api/stream/{episode_id}` — SSE endpoint using `StreamingResponse` with `media_type="text/event-stream"`. Loop: yield events from `episode_events[episode_id]` as `data: {json}\n\n` lines. Each event is `{"type": "step_start"|"step_done"|"step_error"|"log", "step": N, "message": "..."}`. When pipeline completes (status == "done" or "error"), send a final `{"type": "complete"}` or `{"type": "error", "message": "..."}` event and close. Use `asyncio.sleep(0.5)` between poll cycles. Add `Cache-Control: no-cache` and `Connection: keep-alive` headers.

4. `GET /api/images/{episode_id}` — Returns JSON list of image filenames available in the episode_dir. Glob for `scene_*.png` and `thumbnail.png` in episode_dir. Returns `{"images": ["scene_00.png", "scene_01.png", ...]}`. Returns 404 if episode_id unknown.

5. `GET /api/episode/{episode_id}` — Returns full episode state: `{"status", "current_step", "step_states", "scenes" (narrations only), "metadata", "episode_dir"}`. Returns 404 if unknown.

6. `POST /api/upload/{episode_id}` — Calls `podcast.upload()` in a thread, returns `{"video_url": url}` on success or `{"error": msg}` on failure. Use `asyncio.to_thread()` to avoid blocking.

7. `GET /static/{episode_id}/{filename}` — Serves files from the episode_dir. Read the file from `episodes[episode_id]["episode_dir"] / filename` and return with appropriate content-type (image/png for .png, video/mp4 for .mp4). Use `FileResponse`. Returns 404 if file not found.

**Background pipeline function `_run_pipeline(episode_id)`:**
- Redirect stdout to a custom writer that appends log events to `episode_events[episode_id]` with type "log"
- Run steps sequentially:
  - Step 0 (Generate Script): call `podcast.generate_script(topic)`, store scenes
  - Step 1 (Generate Assets): call `podcast.generate_assets()`
  - Step 2 (Render Video): call `podcast.render()`
  - Step 3 (Generate Metadata): call `podcast.generate_metadata()`, store metadata
  - Step 4 (Generate Thumbnail): call `podcast.generate_thumbnail()`
- For each step: append `step_start` event before, `step_done` event after, `step_error` on exception
- On any exception: set status to "error", store error message, break
- On completion: set status to "done"
- Always restore stdout in a finally block

**Stdout capture class `_StdoutCapture`:**
```python
class _StdoutCapture:
    def __init__(self, episode_id, original_stdout):
        self.episode_id = episode_id
        self.original = original_stdout

    def write(self, text):
        self.original.write(text)
        text = text.strip()
        if text:
            episode_events.setdefault(self.episode_id, []).append(
                {"type": "log", "step": episodes[self.episode_id]["current_step"], "message": text}
            )

    def flush(self):
        self.original.flush()
```

```javascript
let episodeId = null;
let pollInterval = null;

async function startGeneration() {
    const topic = document.getElementById('topicInput').value.trim();
    if (!topic) return;

    // Disable input + button
    document.getElementById('topicInput').disabled = true;
    document.getElementById('generateBtn').disabled = true;
    resetUI();

    // POST /api/generate
    const res = await fetch('/api/generate', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({topic})
    });
    const data = await res.json();
    episodeId = data.episode_id;

    // Start SSE
    const evtSource = new EventSource(`/api/stream/${episodeId}`);
    evtSource.onmessage = (e) => {
        const evt = JSON.parse(e.data);
        handleEvent(evt);
        if (evt.type === 'complete' || evt.type === 'error') {
            evtSource.close();
            stopPolling();
            document.getElementById('topicInput').disabled = false;
            document.getElementById('generateBtn').disabled = false;
            if (evt.type === 'complete') {
                document.getElementById('uploadBtn').disabled = false;
                // Fetch final episode data for script + metadata
                fetchEpisodeData();
            }
        }
    };

    // Start image polling
    startImagePolling();
}
```
